# 🌌 Dinesh Naarayanan K | Personal Portfolio Website

A premium, interactive, and fully responsive personal portfolio website showcasing software development, Artificial Intelligence, and Data Science projects. Designed with a custom modern space-theme aesthetic, dynamic particle backgrounds, theme-toggling capabilities (defaulting to light/white theme), and live GitHub contribution calendar integration.

🔗 **GitHub Repository:** [dineshnaarayanan/-Portfolio-Dinesh](https://github.com/dineshnaarayanan/-Portfolio-Dinesh)

---

## 🚀 Live Preview & Aesthetics

The portfolio leverages modern styling guidelines, clean typography (`Outfit`, `Inter`, and `Fira Code` fonts), and fluid transitions:
* **Interactive Hero Header:** Custom profile card linking directly to LinkedIn with hover overlay transitions.
* **Ambient Particles:** A custom-engineered canvas-free stardust particle system that animates in the background.
* **Auto-Typing Subtitles:** Dynamic typewriter animation showcasing various engineering roles.
* **Theme Switching:** Instant toggle between Light Mode (Lavender Starlight) and Dark Mode (Space Void) with persistent `localStorage` saving. Defaults to the clean **Light Theme**.
* **GitHub Integration:** Dynamic, live green contribution calendar fetching real-time active days directly from GitHub.

---

## 🛠️ Built With (Tech Stack)

* **HTML5:** Semantic architecture optimized for SEO.
* **CSS3:** Custom styling using Design Tokens, CSS variables, CSS grid/flexbox layouts, responsive design, and smooth transitions (no heavy frameworks).
* **JavaScript (Vanilla):** Dynamic scroll tracking (ScrollSpy), live project category filtering, custom ambient backgrounds, contact action cards, and theme initialization.

## 💻 Local Setup & Development

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dineshnaarayanan/-Portfolio-Dinesh.git
   cd -Portfolio-Dinesh
   ```

2. **Serve the project:**
   * If you have Python installed, you can start a local server instantly:
     ```bash
     python -m http.server 8000
     ```
   * Or if you use Node.js:
     ```bash
     npx http-server -p 8000
     ```

3. **Open in Browser:**
   Go to `http://localhost:8000/` in your web browser.
